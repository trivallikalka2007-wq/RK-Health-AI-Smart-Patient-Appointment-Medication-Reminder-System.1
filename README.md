# CaptionAI

CaptionAI is an AI-powered application that generates captions using the Groq Large Language Model (LLM).

## Features

- AI Caption Generation
- Groq API Integration
- Flask Backend
- HTML/CSS/JavaScript Frontend

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Groq API

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CaptionAI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=your_api_key
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

## Note

The `.env` file is excluded from GitHub for security.
